HTML <link> Tag

Linking External CSS to Your Webpage

The <link> tag is used to connect external CSS files to your HTML document, allowing you to separate content from design.

Basic syntax:

<link rel="stylesheet" href="styles.css">

Example usage inside the <head> tag:


<head>
  <link rel="stylesheet" href="styles.css">
</head>

Using external CSS helps keep your HTML clean and makes it easier to maintain styles across multiple pages.
