HTML Comments

How to write comments in HTML

HTML comments are used to leave notes in your code that won’t be shown in the browser.

The basic syntax is:

<!-- This is a comment -->

Here’s an example in context:


<!-- This is a heading -->
<h1>Welcome to My Page</h1>

<!-- This is a paragraph -->
<p>This is a simple paragraph.</p>

<!-- This is a link -->
<a href="https://example.com">Visit Example</a>
