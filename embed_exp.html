HTML <embed> Tag

Embedding various external content

The <embed> tag lets you include external resources like multimedia files, PDFs, and more directly in your webpage.

Here are some common examples:

Embedding a PDF file

<embed src="sample.pdf" type="application/pdf" width="600" height="400" />

Embedding an audio file

<embed src="audio.mp3" type="audio/mpeg" width="300" height="50" />

Embedding a video file

<embed src="video.mp4" type="video/mp4" width="640" height="360" />
