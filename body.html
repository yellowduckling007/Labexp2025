HTML <body> Tag

Where the visible content lives

The <body> element contains all the content that is displayed on the webpage, such as text, images, links, and other media.

Everything inside the <body> tag is what users see and interact with when they visit your site.

Common content inside the <body> includes:

  • Headings (<h1> to <h6>)
  • Paragraphs (<p>)
  • Images (<img>)
  • Links (<a>)
  • Lists (<ul>, <ol>)
  • Other HTML elements that structure and display content

Example usage:


<body>
  <h1>Welcome to My Website</h1>
  <p>This is the main content area.</p>
  <a href="https://example.com">Visit Example</a>
</body>
