HTML <head> Tag

What it is and why it’s important

The <head> element is a container for metadata and resources that describe the HTML document. It appears before the <body> tag and does not display content on the page itself.

Content inside <head> helps browsers understand how to process and render your page properly.

Common elements inside the <head> include:

  • <title> — Sets the title shown in the browser tab and search results.
  • <meta> — Provides metadata like character encoding and viewport settings.
  • <link> — Connects external files such as stylesheets and icons.
  • <script> — Loads JavaScript files for functionality.

Example usage:


<head>
  <meta charset="UTF-8">
  <title>My Website</title>
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="styles.css">
  <script src="script.js"></script>
</head>
