HTML <html> Tag

The root element of an HTML document

The <html> tag wraps all the content of an HTML document, including the <head> and <body> sections.

It tells the browser that the document is an HTML document and can include the language attribute (lang) to specify the page language.

Example usage:


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
