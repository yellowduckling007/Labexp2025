HTML <meta> Tag

Metadata and Document Settings

The <meta> tag provides metadata about the HTML document. It is placed inside the <head> section and is not displayed on the page, but helps browsers and search engines understand the content.

Common uses of <meta> tag:

  • Specifying character encoding
  • Setting viewport for responsive design
  • Providing page description for SEO
  • Author information

Example:


<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="A page about the HTML meta tag.">
  <title>Document Title</title>
</head>

Using <meta charset="UTF-8"> ensures your document supports most characters worldwide. The viewport meta tag makes your page mobile-friendly.
